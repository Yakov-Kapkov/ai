# Troubleshooting

When a command produces an unexpected result, consult the dictionary
before reasoning from scratch.

## Workflow

1. Read `references/troubleshooting.md`.
2. Match the observed symptom to a dictionary entry.
3. **Match found** → apply the prescribed fix.
4. **No match** → diagnose and solve normally. The dictionary is a
   shortcut, not a constraint.
